Duality

A GameBoy (Color) port of the GTA San Andreas arcade game Duality.

Controls

On PCs use keyboard input with the keys given above. On mobile devices a touch overlay should automatically appear over the emulator.

Description

Press Left or Right on the title screen to show either the black or white highscores. Press Select to show the about screen and build info. In-game press Start to pause and resume. While paused press Select to return to the menu.

Collect small white spheres to get +5 white score. Collect small black spheres to get +5 black score. The opposite color will reduce your score when collected. Shooting while you have a white score will reduce it by one. Large black holes will attract you and damage your ship when touched. Large white spheres will repel you and replenish your health when touched. Accelerating will reduce your fuel, which will recharge when not accelerating. You can shoot large spheres for +10 points.

For a more detailed description of the original game check out the Duality article on GTA Wiki.

Features

This port has some special features that go beyond the original arcade machine.

  • Multiplayer - Play against a friend with the link cable
  • Game Boy Printer - make hard copies of the score tables

To access multiplayer mode connect two Game Boy systems via link cable, then leave one device on the main menu, while opening the about screen (press SELECT two times) on the other device.

To print the scores open one of the score screens (by pressing LEFT or RIGHT in the main menu), then press SELECT to start printing.
